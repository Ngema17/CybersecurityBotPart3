## Project Overview
This is the final part of the Cybersecurity Awareness Chatbot project. It builds upon Parts 1 and 2 by adding:
- Task Assistant with SQLite database integration
- Cybersecurity Quiz with 12 questions
- Activity Log system
- Natural Language Processing simulation
- Sentiment detection

## Features

### 1. Task Assistant 
- Add cybersecurity tasks
- Set reminders (e.g., "Remind me in 3 days")
- View pending tasks
- Mark tasks as completed
- Delete tasks
- All data stored in SQLite database

### 2. Cybersecurity Quiz 
- 12 questions covering:
  - Password safety
  - Phishing awareness
  - Safe browsing
  - Social engineering
  - Malware
- Multiple choice and True/False questions
- Immediate feedback with explanations
- Score tracking and history

### 3. Activity Log 
- Tracks all user actions
- Timestamps for each action
- Recent actions display
- Database persistence

### 4. NLP Simulation
- Keyword detection for natural language
- Understands variations of commands
- Examples: "add task", "remind me", "view tasks"

### 5. Sentiment Detection 
- Detects worried, frustrated, curious sentiment
- Empathetic responses
- Encouragement for concerned users
